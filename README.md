## 🇬🇧 ENGLISH

### 🥨 About the Project
Designed for anyone who wants to bring a whole universe into the background while studying, reading, or working. 
This project consists of visual themes (each containing multiple media files), curated background music playlists to match, atmospheric ambient sounds, and a suite of customizable time tools:

> ⤷ *Stopwatch & Timer & Alarm & World Clock & Birth Chart*

---

### 🥨 Tech Stack
* **Vanilla JavaScript (ES6+)**: Dynamic theme switching, state management for timers/clocks, and audio mix control without heavy external framework overhead.
* **HTML5 & CSS3**: Responsive UI layout, custom CSS animations, dynamic visual theme transitions, and glassmorphism styling.
* **Web Audio & Media Handling**: Multi-track ambient sound blending and synchronized background music management.

---

### 🥨 Run Locally
If you'd like to run this project on your local machine:

```bash
# Clone the repository
git clone [https://github.com/ietkose/atmosphere-os.git](https://github.com/ietkose/atmosphere-os.git)

# Navigate to the project directory
cd atmosphere-os

# Install dependencies (if applicable)
npm install

# Start the project
npm start
```

## 🇹🇷 TÜRKÇE

### 🥨 Proje Hakkında
Ders çalışırken, kitap okurken veya herhangi bir şey yaparken arka planda bir evreni yaşatmak isteyenlere.
Bu proje; her birinde birden fazla dosya bulunan görsel temalar, bu temaların eşlikçileri müzik tema playlistleri, ambiyans sesleri ve kullanabileceğin saat özelliklerinden oluşuyor:

> ⤷ *Kronometre & Sayaç & Alarm & Dünya Saatleri & Doğum Haritası*

---

### 🥨 Kullanılan Teknolojiler
* **Saf JavaScript (ES6+)**: Dış kütüphane bağımlılığı olmadan dinamik tema değişimi, zaman araçları mantığı ve ses kontrol yönetimi.
* **HTML5 & CSS3**: Esnek (responsive) tasarım, özel CSS animasyonları, akıcı görsel geçişleri ve cam efekti (glassmorphism) arayüz tasarımı.
* **Web Audio & Medya Yönetimi**: Çoklu ambiyans seslerini karıştırma ve senkronize arka plan müzik kontrolü.
