## 🇬🇧 ENGLISH

### 🥨 About the Project
Designed for anyone who wants to bring a whole universe into the background while studying, reading, or working. 
This project consists of visual themes (each containing multiple media files), curated background music playlists to match, atmospheric ambient sounds, and a suite of customizable time tools:

> ⤷ *Stopwatch & Timer & Alarm & World Clock & Birth Chart*

---

### 🥨 Tech Stack
* **Frontend:** HTML5 & CSS3 & JavaScript

---

### 🥨 Run Locally
If you'd like to run this project on your local machine:

```bash
# Clone the repository
git clone [https://github.com/ietkose/atmosphere-os.git](https://github.com/ietkose/atmosphere-os.git)

# Navigate to the project directory
cd atmosphere-os

# Install dependencies (if applicable)
npm install

# Start the project
npm start
```
